Version 1.1.0 also support email + some bug fixes.

**What you need:** Google Chrome.

**Install from a folder (if you received a zip/gzip):**

1. Unzip the folder to a permanent location (e.g. Desktop or Documents). Do not delete it after installing.
2. Open Chrome and go to: **chrome://extensions/**
3. Turn **Developer mode** on (toggle in the top-right).
4. Click **Load unpacked** and select the unzipped folder (the one that contains `manifest.json`).
5. The extension **Page Linkify** should appear and be enabled.

**Use:** Browse any website. Valid phone numbers become clickable and open Loyco Member Insight. You can also right‑click in a phone field and choose **Open in Loyco Member Insight**.

### For Edge users

**What you need:** Microsoft Edge.

**Install from a folder (if you received a zip/gzip):**

1. Unzip the folder to a permanent location. Do not delete it after installing.
2. Open Edge and go to: **edge://extensions/**
3. Turn **Developer mode** on (toggle at the bottom-left).
4. Click **Load unpacked** and select the unzipped folder (the one that contains `manifest.json`).
5. The extension **Page Linkify** should appear and be enabled.

**Use:** Same as Chrome — phone numbers are linkified; you can also use the right‑click menu **Open in Loyco Member Insight** in phone fields.
